# social-cart-ai

An AI-powered community platform for sharing and organizing shopping links and collections from AliExpress, Shein, and more.

## Project Structure

- `src/` - Next.js frontend
- `backend/` - Flask scraping API

## Local Development

### Frontend

```bash
npm install
npm run dev
```

### Backend

```bash
cd backend
python -m pip install -r requirements.txt
python app.py
```

## Environment Variables

Copy `.env.example` to `.env.local` and fill in your own values.
